# RS HealthCare Contact Portal

A healthcare-themed Flask web application to collect and manage patient inquiries.

## Features
- Responsive frontend built with HTML/CSS.
- Admin panel to view all submissions.
- Export to Excel and PDF.
   
## Installation
```bash
git clone https://github.com/your-username/rs-healthcare.git
cd rs-healthcare
pip install -r requirements.txt
python app.py
```
